# Markdown for Agents: Assurator.ch

Assurator helps residents in Switzerland compare insurance products and request personalized quotes.

## Primary actions

1. Compare insurance offers using the form on the homepage.
2. Submit a quote request to `/send_email.php`.
3. Read practical insurance guidance in the blog section.
